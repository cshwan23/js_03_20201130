```
<문5> **Array 객체의 메소드를 정리하면?**

```

push(저장데이터)  
\=> Array 객체 안에 배열변수 만들고 데이터 저장하기

---

join(중간 삽입 문자열)  
\=> 모든 배열변수 안의 데이터를 문자로 붙여 붙인 문자를 리턴하기  
붙일 때 중간삽임문자열이 중간에 삽입된다.

---

sort( )  
\=> 모든 배열변수 안의 데이터를 문자로 취급하여  
오름차순으로 재 배치한다.

---

sort(function(left,right){return left-right;})  
\=> 모든 배열변수 안의 데이터를 숫자로 취급하여  
오름차순으로 재배치한다.

---

sort(function(left,right){return right-left;})  
\=> 모든 배열변수 안의 데이터를 숫자로 취급하여  
내림차순으로 재배치한다.

---

reverse( )  
\=> 모든 배열변수 안의 데이터를 역순으로 재배치 한다.

---

slice(idx1, idx2)  
\=> 인덱스번호 idx1 부터 idx2 이전까지 복사하여  
새로운 Array 객체 생성하여 메위주 리턴하기

---

splice(idx1, n)  
\=> 인덱스 번호 idx1 부터 n 개 오려내서  
새로운 Array 객체 생성하여 메위주 리턴하기

---

pop( )  
\=> 마지막 배열 변수를 오려내서 리턴하기

---

concat( 다른Array객체메위주 )  
\=> 다른 Array객체메위주의 배열변수와 기존 Array의 배열 변수를  
복사한 후 합쳐 새로운 Array 객체 생성하여 메위주 리턴하기  
\`\`\`
